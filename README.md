# Supply Chain Analytics Dashboard

## Project Overview

This Power BI project analyzes supply chain operations, delivery performance, sales, and profitability using a dataset containing 180,519 rows.

The dashboard provides business insights through interactive visualizations, KPI tracking, and operational analysis.

---

# Current Dashboards

## 1. Delivery Performance Dashboard
- Delivery tracking
- Shipping performance
- Delay analysis
- Operational KPIs

## 2. Sales and Profit Dashboard
- Sales analysis
- Profit tracking
- Revenue trends
- Business insights
